# Project Blueprint

## Overview

This project is a simple Next.js application with a basic setup. It includes a homepage, a navbar, and a footer.

## Project Structure

*   `/src/app/page.tsx`: The main page of the application.
*   `/src/components/Navbar.tsx`: The navigation bar component.
*   `/src/components/Footer.tsx`: The footer component.

## Styling

*   **Tailwind CSS**: Used for styling the components.
*   **`globals.css`**: Contains global styles.

## Current Features

*   **Homepage**: Displays a simple welcome message.
*   **Navbar**: Contains links to different pages.
*   **Footer**: Displays copyright information.
